# 🔐 Liquid Vault

A modern offline-first digital vault for storing passwords, private notes, and sensitive information securely on your own device.

No cloud. No accounts. No tracking.

---

## 🚀 Why Liquid Vault?

Most password managers rely on servers, subscriptions, and cloud synchronization.

Liquid Vault follows a different philosophy:

* 100% Offline
* No Account Required
* No Cloud Storage
* Full Data Ownership
* Privacy First
* Open Source

Your data stays on your device. Always.

## ✨ Core Features

### 🔑 Password Manager

Store and manage:

* Usernames
* Passwords
* Website URLs
* Additional Notes

### 📝 Secure Notes

Keep private information organized and protected.

### 💳 Bank Cards

Store:

* Card Number
* Card Holder Name
* CVV
* PIN
* Expiration Date

### 🎲 Password Generator

Generate strong and secure passwords instantly.

### 🔒 Security System

* Master Password
* AES-GCM Encryption
* PBKDF2 Key Derivation
* Auto Lock
* Recovery System
* Secure Backup

---

## 🛡 Privacy Philosophy

Liquid Vault is designed around one simple principle:

**Your Data. Your Device. Your Control.**

No remote servers.

No telemetry.

No data collection.

No third-party access.
