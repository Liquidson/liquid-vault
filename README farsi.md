# 🔐 Liquid Vault

یک گاوصندوق دیجیتال آفلاین برای ذخیره امن اطلاعات محرمانه.

---

## معرفی

Liquid Vault به شما اجازه می‌دهد رمزهای عبور، یادداشت‌های محرمانه و اطلاعات کارت بانکی را مستقیماً روی دستگاه خودتان ذخیره کنید — بدون هیچ سرور، پایگاه داده یا اتصال اینترنتی.

---

## امکانات نسخه 0.1

- ساخت Master Password در اولین اجرا
- ورود با Master Password (هش SHA-256)
- داشبورد با نمایش آمار آیتم‌ها
- جستجوی زنده بین آیتم‌ها
- فیلتر بر اساس دسته‌بندی
- افزودن / ویرایش / حذف آیتم
- پشتیبانی از سه نوع آیتم:
  - **رمز عبور** — نام کاربری، رمز، آدرس سایت
  - **یادداشت** — متن محرمانه
  - **کارت بانکی** — شماره کارت، CVV، PIN، تاریخ انقضا
- آیکون چشم برای نمایش/مخفی فیلدهای حساس (PIN، CVV، رمز)
- Dark Theme کامل
- واکنش‌گرا (موبایل و دسکتاپ)

```
LiquidVault/
├── index.html          # ساختار UI و event listenerها
├── css/
│   └── style.css       # تمام استایل‌ها — Dark Theme
├── js/
│   ├── app.js          # Orchestrator مرکزی — Bootstrap برنامه
│   ├── auth.js         # منطق Master Password و session
│   ├── storage.js      # تنها نقطه دسترسی به LocalStorage
│   └── ui.js           # رندر، فرم‌ها، modal، toast
├── assets/
└── README.md
```

1. پوشه پروژه را دانلود کنید
2. فایل `index.html` را مستقیماً در مرورگر باز کنید
3. تمام شد — نیاز به نصب یا سرور ندارد

---

## معماری

برنامه روی اصل **Separation of Concerns** طراحی شده:

```
UI Layer (index.html + style.css + ui.js)
        ↓ Events
App Layer (app.js) ← Orchestrator
        ↓ Calls
Auth Layer (auth.js) + Storage Layer (storage.js)
        ↓ Persist
LocalStorage
```

هیچ ماژولی مستقیماً با ماژول دیگری صحبت نمی‌کند — همه از `app.js` رد می‌شوند.

## نکته امنیتی

این نسخه یک **MVP** است. داده‌ها در LocalStorage بدون رمزنگاری واقعی ذخیره می‌شوند.

معماری پروژه طوری طراحی شده که در نسخه ۱.۰ بتوان **AES-256-GCM** را فقط با تغییر `storage.js` اضافه کرد، بدون بازنویسی کد.
